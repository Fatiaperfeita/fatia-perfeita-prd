# PRD - Visão Geral

## 1. Visão do Produto

### 1.1 Problema Atual
- Cadastro de pedidos manual no Notion
- Sem controle de custos e insumos
- Dificuldade para calcular precificação
- Sem padronização de receitas

### 1.2 Solução Proposta
Sistema web completo com:
- Cadastro estruturado de insumos
- Cadastro de produtos com receitas e cálculo automático de custos
- Tela de produção com geração de lista de materiais
- Sistema de pedidos com fluxo guiado
- Dashboard organizado
- Integração com automação atual (N8N + WhatsApp)

### 1.3 Benefícios
- Controle preciso de custos
- Padronização de receitas
- Agilidade no cadastro de pedidos
- Melhor precificação
- Menos erros manuais

## 2. Público-Alvo

### Usuários Primários
- **Natália**: Responsável por cadastrar pedidos e gerenciar produção

### Usuários Secundários (Futuro)
- **Clientes**: Acessar sistema para fazer pedidos diretamente

## 3. Sucesso do Produto

### Métricas de Sucesso
- Redução de 50% no tempo de cadastro de pedidos
- Precificação com base em custos reais
- Controle de estoque de insumos
- Relatórios de vendas e lucratividade
